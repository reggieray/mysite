# Statiq Site Manager Skill

This workspace contains a personal developer blog built with Statiq Web running on .NET 8. Use these instructions to immediately understand the project layout, configurations, and scripts without performing codebase discovery.

---

## 📂 Codebase Architecture

### Core Project Files
*   **Project File**: [MySite.csproj](file:///C:/Repos/mysite/MySite.csproj) (standard SDK-style console application referencing `Statiq.Web`).
*   **Application Bootstrapper**: [Program.cs](file:///C:/Repos/mysite/Program.cs) (registers metadata settings, generates XML Sitemaps and Search indices, enables tag-fields indexing, and disables results compression).
*   **Build script**: [localBuild.ps1](file:///C:/Repos/mysite/localBuild.ps1) (helper script to compile site and run preview).

### Content & Layout Templates
*   **Template Source**: [input/](file:///C:/Repos/mysite/input) (contains page views, layouts, and static resources).
*   **Base Layout**: [input/_Layout.cshtml](file:///C:/Repos/mysite/input/_Layout.cshtml) (main shell view, handles dark/light theme initialization script, global `/` keyboard search shortcut, clipboard code copies, and native image lazy-loading).
*   **Post Archives & Tags**:
    *   Home Page list: [input/index.cshtml](file:///C:/Repos/mysite/input/index.cshtml) (uses `ArchiveSources: posts/20*/**/*` to list blog posts and exclude raw HTML report folders).
    *   Tags: [input/tags.cshtml](file:///C:/Repos/mysite/input/tags.cshtml) (compiles posts by tags and links to categories).
*   **Blog Content**: [input/posts/](file:///C:/Repos/mysite/input/posts) (blog posts are organized under folders by year, e.g. `2025/`, `2026/`).
*   **Raw HTML Reports**: [input/posts/html/](file:///C:/Repos/mysite/input/posts/html) (houses NBomber HTML reports and other artifacts linked from posts; excluded from listing feeds and search indices but generated as static files).

## 🛠️ Main Features Reference

### 🎨 Dark & Light Theme Switcher
*   **Styles**: [theme-switcher.css](file:///C:/Repos/mysite/input/assets/css/theme-switcher.css) contains light mode colors, frosted-glass header formatting, gradient banner properties, and copy-button layout rules.
*   **Alignment**: Flexbox constraints are applied to `#header nav` to align search and theme toggles with the menu button on mobile.
*   **Flashes Guard**: Body starts with a `no-transition` class that is removed via JS 100ms after load to prevent color-flashes.

### 🔍 Real-Time Search & Autocomplete
*   **Page**: [input/search.cshtml](file:///C:/Repos/mysite/input/search.cshtml) fetches the metadata dictionary `search.results.json` generated by Statiq.
*   **Features**: Includes real-time autocomplete suggestions, arrow-keys selector navigation, and soft indigo `<mark>` highlighting.

## 💻 Local Commands

To build and launch the local preview server with hot-reloading:
```bash
dotnet run -- preview
```
View the page at **http://localhost:5080**.

## 📦 CI/CD Deployment

Deployments are automated through **GitHub Actions** via Netlify:
*   Workflow config: [.github/workflows/deploy.yml](file:///C:/Repos/mysite/.github/workflows/deploy.yml)
*   Required secrets: `NETLIFY_AUTH_TOKEN` (Personal Access Token) and `NETLIFY_SITE_ID` (Site API ID).
*   Deployment targets: Direct upload of the generated `/output` directory.
